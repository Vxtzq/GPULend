🌟 Overview

💻
Rent GPU
Run your tasks on community GPUs when you need extra memory or compute.
🔗
Share GPU
Offer spare GPU time and earn credits — keep resources flowing in the community.
📊
Manage Jobs
Simple job tracking and credit accounting so you know what ran where.

✨ Features

⚡
Lightweight GUI
Simple interface built with PyQt6 for quick setup and use.
🖥️
Auto GPU Detection
Detects GPUs and available VRAM to recommend where to run jobs.
🌐
Cross-Platform
Windows and Linux support; AppImage / standalone executables included.
🔒
Containerized Execution
Runs user jobs inside Podman containers to reduce risk and isolate workloads.

🛠 Installation

Windows

Download the standalone .exe or run the installer:

git clone https://github.com/Vxtzq/GPULend
install_and_run.bat

Linux

Run the AppImage:

git clone https://github.com/Vxtzq/GPULend
chmod +x GPULend.AppImage
./GPULend.AppImage

Optional: Run from Python (from source):

python -m pip install -r requirements.txt
python app.py

⚡ Quick Start

  1. Launch GPULend.
  2. Ensure Podman is installed and detected.
  3. Choose a remote GPU and submit your job (monitor status in the Jobs panel).
