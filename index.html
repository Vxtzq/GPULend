GPULend 🚀

Free computation for all – rent and share GPUs seamlessly for AI/ML projects.

View on GitHub

🌟 Overview

Rent GPU

Access high-VRAM GPUs from the community.

Share GPU

Offer your own GPU to help others and earn credits.

Manage Jobs & Credits

Track tasks, usage, and rewards with ease.

✨ Features

Lightweight GUI

Portable, fast, and easy to use interface.

Automatic GPU Detection

Detects available GPUs and manages resources automatically.

Cross-Platform

Works on Windows and Linux systems.

Secure Execution

Runs tasks in Podman containers to ensure safety.

Easy Installation

Standalone executable or AppImage – no complex setup required.

🛠 Installation

Windows

Download the standalone .exe or run the installer:

git clone https://github.com/Vxtzq/GPULend
install_and_run.bat

Linux

Run AppImage:

git clone https://github.com/Vxtzq/GPULend
chmod +x GPULend.AppImage
./GPULend.AppImage

Optional: Run from Python:

python -m pip install -r requirements.txt
python app.py

⚡ Quick Start

  1. Launch GPULend.
  2. Ensure Podman is installed and detected.
  3. Offload computation tasks to available GPUs effortlessly.
