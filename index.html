🌟 Overview

💻
Rent GPU
Run tasks on community GPUs when you need extra VRAM or compute.
🔗
Share GPU
Offer your idle GPU to others and earn credits for compute.
📊
Manage Jobs
Simple job tracking, credit accounting, and job history.

✨ Features

⚡
Lightweight GUI
PyQt6-based interface for fast setup and responsive use.
🖥️
Auto GPU Detection
Detects GPUs and available VRAM to suggest targets for jobs.
🔒
Containerized Execution
Runs jobs inside Podman containers to isolate workloads.
🌐
Cross-Platform
Windows & Linux support; AppImage or standalone builds provided.

🛠 Installation

Windows

Download the standalone .exe or run the installer:

git clone https://github.com/Vxtzq/GPULend
install_and_run.bat

Linux

Run the AppImage:

git clone https://github.com/Vxtzq/GPULend
chmod +x GPULend.AppImage
./GPULend.AppImage

Optional: Run from source:

python -m pip install -r requirements.txt
python app.py

⚡ Quick Start

  1. Launch GPULend.
  2. Ensure Podman is installed and detected.
  3. Pick a remote GPU and submit a job; monitor status in the Jobs panel.
